# Letterboxd Inspired Project!

I am a HUGE Letterboxd (and movie fan), so I wanted to see if I could make some sort of a recreation of it.
You can search for movies AND TV shows!


## Description

This project has two pages, a Search page on landing, and a 'Profile' page. You can search for a movie or show, and scroll through to find which one you want to use. Then, once you click on the media, you will be prompted to give a review. The review consists of a comment and a rating from 1-5. After you finish, you click save review. If you navigate to the profile page, you can now see your review on your page. You can also change the background colors of the app! To create this app, I used Swift and a TMDb API.

## Features

While the project is in a very elementary state at the moment, it stands out because of a fun color customizability option- and the fact it includes TV shows, something I had always wished Letterboxd had a section for!

### Dependencies

* To use the app, you will need to access a API key for TMDb.

### API

* TMDb - https://www.themoviedb.org/?language=en-US
